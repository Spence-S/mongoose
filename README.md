## Install

[npm][]:

```sh
npm install @ladjs/mongoose
```

[yarn][]:

```sh
yarn add @ladjs/mongoose
```


## Usage

```js
const Mongoose = require('@ladjs/mongoose');
const mongoose = new Mongoose(agenda, config, logger);
```
