## Install

[npm][]:

```sh
npm install lad-mongoose
```

[yarn][]:

```sh
yarn add lad-mongoose
```


## Usage

```js
const LadMongoose = require('lad-mongoose');

const ladMongoose = new LadMongoose();

console.log(ladMongoose.renderName());
// script
```
